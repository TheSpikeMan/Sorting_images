--------------- WERSJA PL

Program Sorting_images służy do posortowania zdjęć i filmów z danego katalogu, do innego katalogu wg dat. Głównym celem, który stał za jego powstanie stało się sortowanie zdjęć w relacji telefon- komputer. Chciałbym mieć pełną kontrolę nad tym, co i jak robi program, a także zostawić taką możliwość użytkownikowi. 

Dane wejściowe:
- 1. Ścieżka do katalogu, zawierającego zdjęcia i filmy do posortowania.
- 2. Ścieżka wyjściowa, w której zostaną umieszczone posortowane zdjęcia. Należy zaznaczyć, że program sam znajduje lata i miesiące, które charakteryzują zdjęcia i samodzielnie tworzy odpowiednie foldery.
- 3. Ścieżka wyjściowa, w której zapisane zostaną dane o plikach, które zostały wykryte poprawnie jako zdjęcia i filmy, w postaci listy, a także lista tych plików, które nie zostały oznaczone jako zdjęcia i tym samym, nie zostały posortowane.

Program jest w fazie produkcji i nie jest jeszcze przetestowany, dlatego należy go używać z dużą ostrożnością. Celem pośrednim programu jest zaprezentowanie i przetestowanie własnych umiejętności.
W tej chwili wprowadzono możliwość sortowania wg lat i miesięcy. Program w obecnej wersji nie usuwa zdjęć ani nie przenosi ich. Jedyną czynnością jest kopiowanie (i tylko tych zdjęć, których nie ma w danej lokalizacji).

Plany rozwojowe w najbliższym czasie.
- wprowadzenie możliwości sortowania zdjęć wg, zdefiniowanych przez użytkownika, przedziałów czasowych.
- wprowadzenie możliwośći tworzenia logów w formacie .xlsx/.txt zawierających czasy i zadania wykonywane przez program. Wprowadzone zostały przyciski umożliwijące wybranie odpowiedniej opcji, jednak jeszcze nie działają.
- poprawienie aspektu graficznego programu.

---------------- ENG VERSION

Sorting_images program is made to help you sort your images and movies, from one catalogue to other. The main reason of creating this kind of app was copying the pictures and movies in relation: mobile phone- PC. I would like to have full control of what program does and as well as letting the user decice about it.

Input data:
- 1. Path to catalogue, having the pictures and movies to sort.
- 2. Path to catalogue, in which the sorted pictures and movies will by copied to. Program makes the subfolders, having the information about years and months from the filename.
- 3. Path to catalogue, in which the data about pictures and vidoes recognized correctly will be put as well as the data about those not fitting to requirements.

Program is in continuous process of developement and has not been tested yet and therefere has to be used with attention. In current version program does not delete any pictures not movies and does not move them anywhere. The only thing that happens is copying (and only these pictures and videos which are not duplicated).

Development plans:
- introducing possibility to sort by user predefined time frames,
- introducing possibility to create logs in .xlsx/.txt formts containing times and tasks executed by the program. Special option button has been implemented, but there are not working yet.
- improving graphics.
